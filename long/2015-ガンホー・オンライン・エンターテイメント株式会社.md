# ガンホー・オンライン・エンターテイメント株式会社（2015年04月〜2017年09月, 業務委託）

- **職種**: バックエンドエンジニア / マネジメント職種  
- **プロジェクト概要**: PS4向けアクションゲーム(LET IT DIE)のサーバエンジニア  
- **チーム規模**: 50〜100人  
- **役割**: メンバー  
- **利用技術**: PHP, MySQL  

### プロジェクト詳細・担当業務・実績
開発中フェイズに参加し、通信時間を3秒→0.3秒ほどに削減しました。（リポジトリパターンを導入して、データアクセスフローを整備した。）  
リリース後、プロジェクト内で最も早くゲームクリアし、色々と意見させていただきました。
